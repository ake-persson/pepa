# NAME

pepa.conf - pepa configuration file

# SYNOPSIS

/etc/pepa.conf

# DESCRIPTION

Configuration files for pepa.

# OPTIONS

[main]
:   Main section.

basedir
:   Basedir for pepa where input and templates are located, defaults to: /srv/pepa

[host]
:   Host section.

sequence
:   Susbstitution sequence for hosts.
